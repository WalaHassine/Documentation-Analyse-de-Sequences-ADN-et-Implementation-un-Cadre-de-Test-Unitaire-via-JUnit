# Documentation-Analyse-de-Sequences-ADN-et-Implementation-un-Cadre-de-Test-Unitaire-via-JUnit
JUnit 5 est le standard de fait pour l’écriture de tests unitaires en Java, permettant la
création de suites de tests répétables avec des résultats cohérents . Ce framework se compose de
trois modules : JUnit Platform, JUnit Jupiter et JUnit Vintage. Cette architecture offre
une flexibilité pour l’exécution et de la compatibilité. Ce rapport détaille son architecture, son
processus d’installation via Maven et les étapes pour réaliser un premier test, ainsi il spécifie
un exemple de démonstration qui illustre la démarche de test suivie.
